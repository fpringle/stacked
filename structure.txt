




- class Game
    - container class, persistent, singleton
        - => can use encapsulation
    - runs the program
    - member: editor
    - member: map
    - member: display


- class Editor
    - also a singleton, controlled by Game
    - member: internalEditor (codemirror)
    - member: parent Game


- class Map
    - new map for each level, each restart etc.
    - describes the level
    - member: player -> single Player object
